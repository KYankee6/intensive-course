## 1. 만든 이유
> (...) 친구와 여러가지 부트캠프, 국비 학원을 같이 알아봤다. 다양한 교육기관에서 **매우 짧은 시간에 많은 개념**을 교육 시키고 있음을 알게 됐다.
> 이러한 교육 방법이 옳다 그르다 할 입장은 아니어서, "Intensive Course"라는 나만의 교육 과정을 만들어 볼 까 생각 했다. 나의 진도나 노력 등 투자하는 양이 현저히 적어서 라고 생각했기 때문이다.
